## 案例 1：明确违规（保本保息）

**文案：**

> 本产品保本保息，年化收益5%稳稳到手

**合规结论：** ❌ `no`（不合规）

**违规项：** 3 条

| 编号 | 违规类型         | 原因                                                                                 |
| ---- | ---------------- | ------------------------------------------------------------------------------------ |
| V01  | 承诺本金不受损失 | 文案中"本产品保本保息"直接明示保本，违反禁止暗示或明示保本的规定。                   |
| V02  | 承诺确定收益     | 文案中"年化收益5%稳稳到手"属于对收益的确定性承诺，构成违规承诺收益。                 |
| V03  | 使用绝对化用语   | "稳稳到手"属于绝对化、确定性表述，暗示无风险、无波动，构成片面夸大宣传，易引发误导。 |
